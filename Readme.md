Это моя базовая реализация JWT, которая теперь будет кочевать из проекта в проект.

Сделано это чтобы не копировать реализацию из проекта в проект, а сразу иметь готовое решение.

Информация по эндпоинтам:

1. **api/auth/register** - путь для регистрации пользователя  
_Тело запроса_: UserDTO

    Ответ на запрос: Bearer токен

    **Пример запроса**:

    POST localhost:8080/api/auth/register

    {
"email": "newuser",
"password": "mypassword"
}



2. **api/auth/login** - путь для входа пользователяв систему
   
    _Тело запроса_: UserDTO

    Ответ на запрос: Bearer токен

   **Пример**:

   POST localhost:8080/api/auth/login

   {
   "email": "newuser",
   "password": "mypassword"
   }

3. **api/user/info** - запрос пользователя информации о себе (работает только если приложить токен)

    Ответ на запрос: id + email пользователя

   **Пример**:

   GET localhost:8080/api/user/info


**Как приложить токен:** после регистрации/логина вы получите токен. Затем в Postman (например) во вкладке Authorization выбираете тип авторизации через Bearer токен, который затем вставляете в поле.

P. S. DebugController нужен был для дебага (что логично)
